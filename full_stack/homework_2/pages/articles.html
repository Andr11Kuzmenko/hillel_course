Introduction to HTML

HTML (HyperText Markup Language) is the backbone of every web page. It provides the structure and content that browsers display to users. Without HTML, web pages would have no headings, paragraphs, or images.

HTML uses tags, such as <h1> for headings, <p> for paragraphs, <a> for links, and <img> for images. Tags usually come in pairs with opening and closing tags, though some tags are self-closing.

For example, a simple HTML structure looks like this:

<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello World</h1>
    <p>This is my first webpage.</p>
  </body>
</html>

HTML also supports semantic elements like <header>, <footer>, <main>, and <section> to improve accessibility and SEO. Learning HTML is the first step to becoming a web developer.

CSS Basics

CSS (Cascading Style Sheets) is used to style HTML elements. While HTML provides structure, CSS controls appearance, layout, and visual design. With CSS, you can change colors, fonts, spacing, and create animations.

CSS works with selectors to target HTML elements. For example:

p {
  color: blue;
  font-size: 16px;
}

This will make all paragraph texts blue and 16px in size. CSS can be written inline, internally in the <style> tag, or externally in a separate stylesheet.

CSS also allows flexible layouts using properties like flexbox and grid, which are essential for modern web design. Media queries help make your designs responsive so they adapt to different screen sizes.

Some common CSS properties include:

  • Background colors and gradients
  • Margins, padding, and borders
  • Font families, sizes, and weights
  • Animations and transitions

JavaScript Fundamentals

JavaScript is the programming language that adds interactivity to web pages. With JavaScript, you can respond to user actions, manipulate page content, and build dynamic interfaces.

For example, you can use JavaScript to show an alert when a user clicks a button:

<button onclick="alert('Hello!')">Click Me</button>

JavaScript uses variables, functions, loops, and conditionals to control behavior. You can manipulate the DOM (Document Object Model) to change elements, text, or styles dynamically.

Popular uses of JavaScript include:

  • Form validation
  • Interactive galleries and sliders
  • Fetching data from servers (AJAX / Fetch API)
  • Single Page Applications (SPAs)

Learning JavaScript is essential for front-end development and allows you to create interactive websites and web applications.

Responsive Design

Responsive design ensures that websites look and function well on all devices, from large desktop monitors to small mobile screens. Without responsive design, a website may appear broken or unreadable on phones.

Techniques for responsive design include:

  • Flexible layouts with percentages instead of fixed pixels
  • Media queries to apply different styles at different screen widths
  • Flexible images and videos that scale automatically
  • Using CSS frameworks like Bootstrap or Tailwind for responsive grids

Example of a media query:

@media (max-width: 600px) {
  .container {
    flex-direction: column;
  }
}
